# Architecture Language

> Auto-generated by CapsuleMap.
> Project: `CapsuleMap`

Use these words consistently when handing work to a coding agent.

## Handoff Pack
- The generated `docs/ai/*` files that summarize what to read, what may break, and what to test.

## Capsule
- A concise current-state summary. It should orient the next agent before it scans the whole repo.

## Impact Map
- File-level imports, reverse imports, nearby tests, and risk rank.

## Test Map
- Source-to-test and test-to-source relationships. Use it to pick focused tests before broad regression.

## Local Judge
- Optional local-model or heuristic triage. It can recommend reads and task type, but it is not the source of truth.

## Deletion Test
- If a module vanished, would complexity disappear or spread into callers? Good modules hide meaningful complexity.
